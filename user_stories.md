# Smart Clinic Management System - User Stories

## Doctor User Stories

### View Appointments
As a doctor, I want to view all of my scheduled patient appointments so that I can manage my daily schedule effectively.

### View Patient Information
As a doctor, I want to view patient details associated with my appointments so that I can prepare for consultations.

### Manage Prescriptions
As a doctor, I want to create and manage prescriptions for my patients so that patients can receive appropriate treatment.

## Patient User Stories

### Search for Doctors
As a patient, I want to search for doctors by name and specialty so that I can find the appropriate doctor for my healthcare needs.

### Book Appointments
As a patient, I want to book an appointment with an available doctor so that I can receive medical consultation.

### View Appointments
As a patient, I want to view all my booked appointments so that I can keep track of my consultations.

### View Prescriptions
As a patient, I want to view prescriptions provided by my doctor so that I can follow the prescribed treatment.

## Admin User Stories

### Add Doctors
As an admin, I want to add new doctors to the Smart Clinic Management System so that they can provide healthcare services to patients.

### Manage Doctors
As an admin, I want to view and manage registered doctors so that doctor information remains accurate and up to date.

### Manage Patients
As an admin, I want to manage patient records so that the clinic maintains accurate patient information.

### Manage System
As an admin, I want to manage the Smart Clinic Management System so that doctors, patients, and appointments can be administered efficiently.
